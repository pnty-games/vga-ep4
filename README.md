# Everything is a Panel: Building a DOS UI System | Video Game Anachronism Ep. 4

This is the accompanying code for [Episode 4](https://youtu.be/QdM1RU0gCRo) of the Video Game Anachronism series.

As before, everything is cumulative from VGA-EP1 + VGA-EP2. This code base is updated to include VGA-EP4's code as well:
This includes:
  • [VGA Episode 1 Code](https://github.com/pnty-games/vga-epi1)
  • [VGA Episode 2 Code](https://github.com/pnty-games/vga-epi1))
	•	PixelBuffer (16-bit friendly rendering buffer)
	•	Sprites built on top of PixelBuffers
	•	Basic geometry with Rect
  • Text Rendering + BIOS Fonts
	•	Everything is a Panel! - A simple UI system
